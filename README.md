Simple API client for [Dead Man's Snitch](https://deadmanssnitch.com)

![Snitches get Stitches](doc/get_them_stitches.jpg)

## Basic Usage

Notify Dead Man's Snitch that a thing happened:
```ruby
Snitcher.snitch("c2354d53d2")
```
